# EnglishStudy - 英语学习H5页面
一个完整的PC端英语学习网站，包含听、说、读、写、单词、语法六个模块。
## 功能特性
- 🎧 **听力模块**：支持音频上传和播放
- 🎤 **口语模块**：支持录音和音频上传
- 📖 **阅读模块**：支持文本、PDF、图片上传
- ✍️ **写作模块**：在线写作练习
- 📚 **单词模块**：单词学习和管理
- 📝 **语法模块**：语法规则学习
- 📊 **单词统计**：自动统计整个站点的英文单词量
## 技术栈
- HTML5
- CSS3 (响应式设计)
- JavaScript (原生，无框架依赖)
## 部署
1. 将代码推送到 GitHub 仓库
2. 在仓库 Settings → Pages 中启用 GitHub Pages
3. 选择 main 分支和 /(root) 文件夹
4. 访问 `https://shency999-ux.github.io/EnglishStudy/`
## 使用说明
1. 克隆仓库到本地
2. 在浏览器中直接打开 `index.html` 即可使用
3. 所有功能都在客户端运行，无需服务器
